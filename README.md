# qualifications_helper

> 취업 준비생을 위한 **채용공고 + 자격증 시험 일정 통합 관리 PWA**

---

## 왜 만들었나

직장을 다니면서 개발자 취업을 준비하다 보면 채용 공고 마감일을 놓치거나, 자격증 시험 접수 기간을 모르고 지나치는 일이 생긴다. 이 앱은 두 가지 일정을 한 곳에서 관리하고, 모바일에서도 빠르게 확인할 수 있도록 PWA로 제작한다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 자격증 시험 일정 관리 | 접수일 · 시험일 · 결과발표일 등록 및 D-day 표시 |
| 채용공고 관리 | 회사 · 포지션 · 마감일 · 링크 · 메모 등록 |
| 홈 대시보드 | 임박한 일정 상위 노출, D-day 뱃지 |
| PWA 지원 | 홈 화면 추가, 오프라인 동작 |
| 브라우저 푸시 알림 | 마감 N일 전 알림 (예: 3일, 7일 전) |

---

## 기술 스택

- **JavaScript** (Vanilla JS)
- **PWA** — `manifest.json` + `service-worker.js`
- **CSS** — 컴포넌트별 파일 분리, 모바일 퍼스트 (360px 기준)
- **로컬 스토리지** — 초기 데이터 저장 (추후 IndexedDB 전환 예정)

## UI 구조 (와이어프레임)

```
┌─────────────────────────────┐
│  📋 qualifications_helper    │  ← 헤더 (앱 이름 + 설정 아이콘)
├─────────────────────────────┤
│  [ 대시보드 | 공고 | 자격증 ]  │  ← 탭 네비게이션
├─────────────────────────────┤
│                              │
│  🔔 임박한 일정               │  ← DashboardSection
│  ┌──────────────────────┐   │
│  │ [D-3] 정보처리기사 실기  │   │  ← ExamItem (카드)
│  └──────────────────────┘   │
│  ┌──────────────────────┐   │
│  │ [D-7] 카카오 백엔드 마감│   │  ← JobCard (카드)
│  └──────────────────────┘   │
│                              │
├─────────────────────────────┤
│  [+ 일정 추가]               │  ← FAB (플로팅 액션 버튼)
└─────────────────────────────┘
```

### 공고 카드 (JobCard)
```
┌──────────────────────────────┐
│ 카카오          [D-7] 마감    │
│ 백엔드 개발자 (3년 이상)       │
│ 마감: 2026-06-23             │
│ 🔗 공고 링크   📝 메모 보기    │
└──────────────────────────────┘
```

### 자격증 카드 (ExamItem)
```
┌──────────────────────────────┐
│ 정보처리기사       [D-3] 시험  │
│ 접수: ~2026-05-30            │
│ 시험: 2026-06-19             │
│ 결과: 2026-07-11             │
└──────────────────────────────┘
```

## 시작하기

```bash
# 로컬 서버 실행 (PWA Service Worker는 HTTPS 또는 localhost 필요)
npx serve .
# 또는
python -m http.server 3000
```

브라우저에서 `http://localhost:3000` 접속 후 모바일 크기(360px)로 확인.
